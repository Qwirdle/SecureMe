Tools and Apps

  • haveibeenpwned: Very useful website that allows one to check if ones password has been leaked.

  • DuckDuckGo: Very secure web browser, but not over top, allowing it to still be very useable.

  • Proton VPN: Great free VPN, if you need to use one for any projects/general safety.

  • Privacy Badger: Browser extension that blocks invisible web trackers.

  • AdBlock: Great ad blocker, good for everyday experience and blocking malicious ads.

  • RandomKeygen: Good for generating super strong passwords, also good for generating keys for other applications.
